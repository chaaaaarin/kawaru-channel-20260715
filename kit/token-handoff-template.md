# 🎁 トークン切れ引き継ぎ書テンプレート

GPT-5.6/Codexの利用枠（5時間・週次）が切れても、Claude Codeなど別のAIツールにそのまま貼り付けて続きから再開できる「引き継ぎ書」の型です。

**受け取り方**: 下のコードブロック右上のコピーボタンでコピーするか、このファイルをそのままダウンロードしてください。

**使い方**: 長引きそうなタスクだと感じた時点で、各項目の〔　〕を埋めながら書き進めてください。利用枠が切れた／別のツールに移りたくなったタイミングで、そのまま貼り付けて「この続きから再開してください」と伝えるだけで引き継げます。

```text
【トークン切れ引き継ぎ書】

# 目的（このタスクで最終的に達成したいこと）
〔1〜2文で、成果物・完了条件を書く〕

# ここまでの進捗
- 〔完了したこと1〕
- 〔完了したこと2〕
- 〔現在どこで止まっているか〕

# 次にやること（優先順）
1. 〔直近でやるべきこと〕
2. 〔その次〕

# 決定済みの事項（やり直さなくていいこと）
- 〔採用した設計・方針とその理由〕
- 〔検討したが不採用にした選択肢と理由〕

# 触ったファイル・参照した情報
- 〔ファイルパス／URL／ドキュメント名〕

# 避けるべきこと・注意点
- 〔ハマった落とし穴・繰り返さない失敗〕
- 〔変更してはいけない箇所〕

# 引き継ぎ先への一言
このドキュメントを読んだら、上記の続きから作業を再開してください。
不明点があれば、作業を進める前に確認してください。
```

**ポイント**: 「切れてから慌てて書く」のではなく、タスクを始める時点でこの型を頭に置いておくと、途中経過のメモがそのまま引き継ぎ書になる。移行先が変わっても（Claude Code・別のChatGPTセッションなど）同じ型で通用する。
