節約テクニック①〜⑤

原則はひと言：「計画は賢いモデル、実行は安いモデル」。設計判断だけ上位モデルに任せ、実装・反復は下位モデルに渡す。

1

モデル・Effortの使い分け

簡単な質問はチャットで済ませ、簡単なタスクはLuna、日常業務はTerra、複雑・高価値タスクだけSolに絞る

✅ 公式ガイダンス

2

Outcome-Firstプロンプト

手順書型をやめ、成果定義と完了条件だけを書く。経路はモデルに委ねる

✅ トークン-41〜66%

3

キャッシュ／Batch API

固定部分を先頭に置きキャッシュを効かせる。非同期処理はBatch APIへ

✅ 読み取り90%引き／Batch 50%引き

4

AGENTS.mdでsubagent対策

「指示なしでsubagentを生成しない」を1行追記し、Solの過剰な自動生成を抑える

🔶 実運用レポート

5

トークン切れの引き継ぎ書

利用枠が切れる前に進捗・次にやることをまとめておき、別ツールへそのまま貼って再開する

🔶 実践Tips

もう一段具体的に：使い分けの目安
  • Terra Ultra級の性能が要らないなら、常に高エフォート設定のLunaの方が同等以上でより安い
  • Sol Highより下は検討せず、高エフォートのLunaを使う
  • Sol Extra Highは検討せず、Terra Ultraを使う
  • Sol Ultraの追加コストは、Maxほどの価値がないことが多い

🔶 分析: X @rasbt（Sebastian Raschka氏）。公式ベンチマーク「Artificial Analysis Coding Agent Index」のデータをもとにした個人の見解

Effortレベルと速度モード
Effort使いどころ
Light（Low）素早く完結する軽いタスク向け ✅
Medium標準の初期値。まずここから ✅
Highより複雑な問題に、深く考える時間を与える ✅
Extra HighHighよりさらに深い推論が必要なとき ✅
Max最も難しい問題に最大限の推論時間を与える ✅
Ultra最大推論＋複数エージェントへの自動タスク委任 ✅

Ultraは「設定」ではなくEffortの最上位段階そのもの ✅（公式）。ただしCodex実装のバグで委任先のsubagentが暴走しやすく、直るまでは避けるのが無難 🔶。Fast modeはクレジット消費が2.5倍、GPT-5.6は自走力が高くFast併用だと1メッセージで利用枠の40%を消費することも 🔶。しばらくFast modeは避けたい。

注意点・FAQ

⚠️ 安さだけ追うと起きること

  • SWE-Bench ProではFable 5がSolを上回る報告もあり、ベンチマークによって強みが変わる 🔶
  • ティア×エフォートは両方の軸で試算するのが安全 🔶
  • Fable 5は週次利用が50%制限される時期があり、GPT-5.6登場で緩和・リセットされた形跡がある 🔶
  • 利用制限は変わりやすい：2026年7月12日、OpenAIが5時間制限を一時撤廃し利用枠をリセット（週次上限は継続）。公式アカウントを定点観測すると良い ✅
FAQ
Terraだけで十分ですか？
日常業務なら十分なケースが多い。複雑・高価値なタスクだけSolに絞るのが基本方針
無料版でSolは使えますか？
通常チャットでは使えない。Solをチャットで使うならPlus以上が必要 🔶

まとめ・キットの受け取り方

節約は「使い分け・削る・回す」の3つで決まる。Sol/Terra/Lunaをタスクの難易度で使い分け、プロンプトはOutcome-First型へ削り、キャッシュとBatch APIで請求額を抑える。

今日からやること
  • 簡単な質問はワーク/Codexを開かずチャットで済ませる
  • Effortはmedium/highから始める。Ultraは今は避ける
  • Fast modeはいったんオフにする
  • AGENTS.mdに「指示なしでsubagentを生成しない」を1行追記する
  • 長引きそうなタスクは早めに「引き継ぎ書」の型を用意する
  • 定型・スケジュールタスクはGPT-5.6以外の軽量モデルに固定する
  • 利用枠のリセット・制限緩和の公式アナウンスを定期チェックする
🎁 キットの中身（3点セット）
  • モデル選定チェックリスト（Sol/Terra/Luna×エフォート×速度の使い分け早見表）
  • Outcome-Firstプロンプトテンプレート（成果定義・完了条件の型）
  • トークン切れ引き継ぎ書テンプレート（別ツールへそのまま貼って再開できる型）

受け取り方はコピペ・ダウンロードどちらでも可能。詳しくは プレゼントキット から。
